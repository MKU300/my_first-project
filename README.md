# Deep Neural Network – Application

This project implements a deep neural network from scratch using NumPy.

## Project Structure
- Files/W4A2/Deep Neural Network - Application.ipynb – Main notebook
- dnn_app_utils_v3.py – Helper functions
- datasets/ – Data files
- images/ – Visualizations

## Concepts Covered
- Forward propagation
- Backpropagation
- Zero, Random, and He initialization
- Binary classification

## Tech Stack
- Python
- NumPy
- Jupyter Notebook
